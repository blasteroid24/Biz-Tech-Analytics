##  Getting Started
```bash

For LOcal run without docker

# Install dependencies
npm install

# Setup Env Variables
Copy Schema from .env.local to .env 
Set Variables according to your enviornment.

# Push schema
npm run db:push 

# Seed the admin credentails and workers data
npm run db:seed

# Start server
npm run dev
```
API Documentation available at `/api-docs`.
